AZ Fish Tracker

Developed a responsive frontend with plans to build out the backend for an app that provides information on fish species in Arizona and the records caught. The app, AZ Fish Tracker, connects to an api to grab real-time forecast data for the lakes in AZ, and uses web storage to store user information before the user system is set up. The app also utilizes multiple jQuery plugins to improve the user experience. The backend will allow users to signup and track their own records in Arizona fishing.

Gaming Industry Database Project

Designed a database using Object Role Modeling that our team then integrated in it into a SQL database (Microsoft SQL Server) and performed queries to answer our team's questions. We then transitioned the database, using Docker containerization into a NoSQL database (Couchbase), in order to compare the use of SQL and the alternate structure of JSON and SQL++ in Couchbase.
